scicos_snorm

Scicos S-Norm Block.

Description

This block compute the following fuzzy S-Norm class:

  • Dubois

  • Yager

  • Drastic Sum

  • Einstein Sum

  • Algebraic Sum

  • Maximum

The user can select the S-Norm class and set the parameters in the usual set sequence.
