scicos_tnorm

Scicos T-Norm Block.

Description

This block compute the following fuzzy T-Norm class:

  • Dubois

  • Yager

  • Drastic Product

  • Einstein Product

  • Algebraic Product

  • Minimum

The user can select the S-Norm class and set the parameters in the usual set sequence.
