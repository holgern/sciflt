about_FLT

About Scilab Fuzzy Logic Toolbox

Description

sciFLT is a Fuzzy Logic Toolbox designed to scilab, using scilab, FORTRAN and C code.

  • Types of Fuzzy Logic systems

    At this stage, sciFLT work with Mamdani and Takagi-Sugeno Fuzzy logic systems.

    Scicos Support

    Member functions, S-Norm, T-Norm, Complement and Fuzzy Logic System (fls) are supported. Also include a palette.

    Internal Editor

    sciFLT have an embedded editor written in TCL/TK.
