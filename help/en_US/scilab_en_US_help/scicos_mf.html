scicos_mf

Scicos Member Function Block.

Description

This block compute the followin member functions:

  • Triangular

  • Trapezoidal

  • Gaussian

  • Extended Gaussian

  • Generalized Bell

  • Sigmoidal

  • Product of two Sigmoidal

  • Absolute difference of tho Sigmoidal

  • S-Shaped

  • Z-Shaped

  • Pi-Shaped

The user can select the member function type and set the parameters in the usual set sequence.
